📜 Commandes principales

/serverinfo - Affiche les informations détaillées du serveur
/ping - Tester la latence du bot

⚙️ Autres commandes (Préfixe : !)

!member - Le bot dit le nombre actuel de membres sur le serveur

⚖️ Politique de Confidentialité

Luluhoo Bot s'engage pour la protection de votre vie privée :

  • Collecte minimale : Nous ne stockons que les identifiants techniques indispensables (User ID, Server ID).
  • Zéro espionnage : Aucun message privé n'est lu, analysé ou enregistré.
  • Droit à l'oubli : Toutes les données liées à un serveur sont automatiquement supprimées si le bot est retiré.

Pour toute question ou support, n'hésitez pas à contacter le développeur sur Discord.
