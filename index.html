📜 Commandes principales

⚖️ Politique de Confidentialité

Luluhoo Bot respecte votre vie privée :

  • Nous ne collectons que les IDs nécessaires (User ID, Server ID).
  • Aucun message privé n'est lu ou enregistré.
  • Les données sont supprimées dès que le bot quitte le serveur.

Pour toute question, contactez le développeur sur Discord.
